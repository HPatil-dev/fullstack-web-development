01 - HTML Foundation
This folder contains the structural building blocks for the web projects completed in the first modules of the bootcamp.


📁 Sections & Exercises


** Text & Structure **
2.1 Heading Element: Practicing the hierarchy of <h1> through <h6>.

2.2 Paragraph Element: Using the <p> tag for body text.

2.3 Void Elements: Implementation of self-closing tags like <br /> (line break) and <hr /> (horizontal rule).



** Organization & Lists **
3.0 List Elements: Creating Ordered (<ol>) and Unordered (<ul>) lists.

3.1 Nesting & Indentation: Learning how to properly nest list items (<li>) inside parent tags to maintain clean, readable code.



** Hyperlinks & Media **
3.2 Anchor Elements: Creating links using the <a> tag and understanding the href attribute.

3.3 Image Elements: Embedding images using the <img> tag and defining src and alt attributes.



** Capstone Projects **
2.4 Movie Ranking Project: Using headings and lists to organize a "Top 3" list.

3.4 Birthday Invite Project: Combining text and images to create a simple invitation page.

4.3 HTML Portfolio: A master project that links all the above work into a single navigation hub.
