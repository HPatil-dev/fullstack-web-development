# Full Stack Web Development Projects

A structured collection of projects built while learning full-stack web development through Angela Yu’s course on Udemy.

This repository focuses on hands-on implementation of core technologies including HTML, CSS, JavaScript, Node.js, Express, React, APIs, EJS, Git, GitHub, and databases.

---

## 📂 Repository Structure

Each section is organized based on course progression:

```
01 - HTML Basics
02 - CSS Fundamentals
03 - Bootstrap
04 - JavaScript
...
```

Each folder contains projects and exercises related to that topic.
